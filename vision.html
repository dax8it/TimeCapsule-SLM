🌟 Upcoming Showcase Projects

This Week's Goal: Release TimeCapsules on 2 Amazing Projects

⚡

GPT-2 Speed Run (modded-nanogpt)

Ultra-Fast Transformer Training Optimization

A collaborative optimization effort training GPT-2 124M to target validation loss (~3.28 on FineWeb) as fast as possible using 8× NVIDIA H100 GPUs. Achieved record time of ~3 minutes vs. original 45 min baseline.

🔑 Core Innovations:
• Architectural advances: RoPE, QK-Norm, ReLU², untied heads, embedding shortcuts, FlexAttention
• Muon optimizer: Custom optimizer with Newton-Schulz iteration (~35% faster throughput)
• Systems optimization: FP8 head computation, token prefetching, optimized batching
• Community-driven: Open discussions and experimental tweaks via speedrun ideas board
Timeline: This Week
Focus: Squeezing every drop of performance through clever model design and system engineering
🎮

Pure RL Project (avataRL)

From-Scratch Reinforcement Learning Research

Exploring pure reinforcement learning to train agent behaviors from scratch, taking inspiration from AvataRL-style multi-agent environments with token-inspired agent methods and academic-style investigation.

🔎 Key Focus Areas:
• Pure RL experiments: On-policy training and distillation using KL-divergence
• Research-driven goals: Grant-backed compute, collaborative documentation
• Open methodology: Design docs like THINKING.md for experimental transparency
• Novel architectures: Token-based agent systems with evolving methodology
Timeline: This Week
Focus: Pure-research playground for training RL agents with documentation and experimentation
